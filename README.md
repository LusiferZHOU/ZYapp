这个项目是我在休息之余，使用vue框架，后端是爬虫，由于没有上传到服务器，而是和前端一起发到了GitHub上，需要克隆下来在本地使用

### 步骤

* 需要在jiandan-bend/jiandan-fend中分别在命令行界面使用一个语句

    > npm install

* 环境初始化完成后

    1. 进入煎蛋后端段子，在此处进入命令行界面，输入
    
        > npm start
    
    2. 在进入jiandan-fend前端，同样进入命令行界面，键入
        
        > npm run dev
